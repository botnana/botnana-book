程式開發界面

Botnana Control 內建伺服器。伺服器提供以下程式開發界面：

  • JSON API：程式可以使用 JSON 格式和 Botnana Control 溝通。適用於各種 支援 JSON 及 Websocket 的語言，如 Java、C#、C++、Python、Ruby 等。
  • Javascript API 提供 Javascript 程式和 Botnana Control 溝通的界面。
